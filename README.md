camel-web-poc
=============

- 	Simple web application POC that uses Apache Camel, ActiveMQ and Spring Framework

- 	It demonstrates simple transformation, content based routing and use of JMS and file endpoints

- 	Have been tested with JBoss AS 5x

- 	Dependency camel-extra may be required to be installed locally. 
	https://code.google.com/a/apache-extras.org/p/camel-extra/?redir=1

- 	Project was enhanced from example jboss artifact that is available when you install 
  	jboss-tools in eclipse.
